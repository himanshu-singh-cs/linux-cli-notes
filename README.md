## 📁 File Management

| Command | Description |
|---------|-------------|
| pwd | Print current working directory |
| ls | List files and directories |
| cd | Change directory |
| mkdir | Create a directory |
| touch | Create a file |
| cp | Copy files |
| mv | Move or rename files |
| rm | Remove files |
| cat | Display file content |
| head | Show first 10 lines |
| tail | Show last 10 lines |
